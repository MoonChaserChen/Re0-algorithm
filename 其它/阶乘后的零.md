# 阶乘后的零
给定一个整数 n，返回 n! 结果尾数中零的数量。

```
示例 1:
    输入: 3
    输出: 0
    解释: 3! = 6, 尾数中没有零。
    
示例 2:
    输入: 5
    输出: 1
    解释: 5! = 120, 尾数中有 1 个零.
    
说明: 你算法的时间复杂度应为 O(log n) 。
```

来源：[LeetCode](https://leetcode-cn.com/problems/factorial-trailing-zeroes)

## 解答
结果： `c = n // 5 + n // 25 + n // 125 + ...`
```python
def trailing_zeroes(n: int) -> int:
    c = 0
    while n > 0:
        n //= 5
        c += n
    return c
```
