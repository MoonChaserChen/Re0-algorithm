# 位操作技巧
## 乘2
```
n >> 1
```
## 除2取整
```
n << 1
```
## 获取最低位
```
n & 1
```
## 获取X的第n低位
```
法一：(x>>(n-1)) & 1
法二：(x & (1<<(n-1)))>>(n-1)
```
## 最后一个1变0
最后一个1的位变成0，其它位不变
```
n & (n - 1)
```
## 异或
```
a ^ 1 = ~a
a ^ a = 0
a ^ b = b ^ a
a ^ 0 = 0 ^ a = a
若 a ^ b = c, 则 a = b ^ c, b = a ^ c

a & 0 = 0
a & 1 = a
```
## 交换两个整数变量的值
```
A = A ^ B
B = A ^ B    [B = (A ^ B) ^ B]
A = A ^ B    [A = (A ^ B) ^ (A ^ B) ^ B]
```
## 获取最右侧的1
```
x & -x
```
