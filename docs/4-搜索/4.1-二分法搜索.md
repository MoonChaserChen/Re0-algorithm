# 二分法搜索
对有序数组中是否存在某个元素进行搜索，从最中间元素开始搜索，每次排除一半

```python
def bi_search(arr, e):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == e:
            return mid
        elif arr[mid] < e:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
```
